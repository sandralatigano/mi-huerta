# Mi Huerta 🌿

Sistema integral de gestión, bitácora y control automatizado para huertas domésticas y comunitarias. Desarrollado con una arquitectura limpia y organizada en módulos lógicos.

## 🚀 Tecnologías Utilizadas
* **Backend:** PHP 8.2 (Programación Orientada a Objetos)
* **Base de Datos:** MySQL (Estructura relacional e índices optimizados)
* **Frontend:** HTML5, CSS3, Bootstrap 5
* **Interactividad:** JavaScript moderno y alertas dinámicas con SweetAlert2

## 📂 Estructura del Sistema (Componentes Lógicos)

El sistema se encuentra completamente documentado y dividido en 6 capas operativas:

1. **Núcleo y Configuración (Core):** Gestión de la infraestructura básica y conexión principal a la base de datos `db_huerta`.
2. **Autenticación y Seguridad:** Control de acceso seguro, manejo de sesiones activas y restricción de rutas.
3. **Gestión de Plantas:** Módulo central para la administración de cultivos (variedades, épocas de siembra y ubicaciones).
4. **Seguimiento y Reportes:** Bitácora diaria de cuidados (riego, abono, poda, cosecha) y estadísticas visuales del estado de la huerta.
5. **Gestión de Usuarios y Perfiles:** Control de acceso basado en roles (Administradores y Operadores) con historiales de auditoría.
6. **Interfaz y Estética (UI/UX):** Layouts responsivos y hojas de estilo optimizadas (incluyendo formato para impresión de reportes).
